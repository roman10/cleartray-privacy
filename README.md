# Cleartray Privacy Policy

Source of the privacy policy hosted at <https://roman10.github.io/cleartray-privacy/> and linked from the Cleartray Android app's Play Store listing.

- `index.html` — rendered policy served by GitHub Pages.
- `PRIVACY.md` — the canonical Markdown source.

When the policy changes, edit `PRIVACY.md`, mirror the change into `index.html`, bump the **Effective date**, and push. The URL stays stable across updates.
